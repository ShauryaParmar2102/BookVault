## 📚 BookVault

BookVault is a full-stack web application that helps users manage and track their personal reading list. Users can add books with details like title, author, genre, description, rating, and reading status, then view, search, and filter them through a clean and responsive interface.

## ✨ Features

➕ Add new books with full details

📖 View all books in a clean card layout

🔍 Search books by title

🎯 Filter books by status:

- Not Started
  
- Currently Reading
  
- Finished
  
⭐ Rate books (1–5 system)

⚡ Dynamic UI updates (no page reloads)

🎨 Simple and clean design with hover effects

🔗 Backend integration with API (/books)

## 1️⃣ Download the Project

- Click the green Code button on GitHub

- Select Download ZIP

- Extract the ZIP file on your computer

- Open the project folder in Visual Studio Code

2️⃣ Install Dependencies

- Open a terminal inside the project folder (where package.json is located):

- npm install

This installs all required packages and creates the node_modules folder.

3️⃣ Start the Server

Run:

npm run start

4️⃣ Open the App

Open your browser and go to:

http://localhost:8000
